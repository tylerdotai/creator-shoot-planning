# Creator Shoot Planning Toolkit

> **Stop freezing mid-shoot. Stop second-guessing angles. Stop wondering if your audio is clean.**

A skill + template toolkit for content creators and videographers who shoot their own content. Plan any shoot in a repeatable, structured process — then execute on location without mid-shoot panic.

Built for any camera, any location, any vibe.

---

## What This Is

A toolkit you use every time you have a shoot. Not a one-time course — a repeatable system.

**What's in the box:**

| What | What it does |
|------|-------------|
| `docs/` | 8 skill modules — one for each part of the planning process |
| `shoot-template/` | Blank template to copy for every shoot |
| `skills/` | Execute card — the day-of checklist |
| `templates/` | Skill template for adding your own techniques |

---

## The 8 Toolkit Modules

```
01-brief.md     → Capture the brief (what are you shooting, where, what vibe)
02-research.md  → Scout the location (time of day, crowds, audio environment)
03-style.md     → Define the visual aesthetic (what to steal from your refs)
04-shot-list.md → Build the shot list (exact framing, where to stand, what to say)
05-angles.md    → Angle confidence system (three fallbacks so you never freeze)
06-audio.md     → Audio decision tree (which mic for which scenario)
07-anxiety.md   → Anxiety protocol (what to do when each trigger hits)
08-execute.md   → Day-of execution + post-shoot debrief
```

---

## The Core Problem This Solves

Content creators freeze on location. Not because they can't shoot — because they haven't planned.

**The freeze loop:**
1. Get to location
2. Don't know what to shoot first
3. Second-guess every angle
4. Worry the audio isn't working
5. Feel everyone watching
6. Pack up early, go home with footage that doesn't match what was in your head

**The toolkit breaks that loop:**
- You arrive with a shot list — exact framing, exact positions, exact things to say
- You arrive with angle fallbacks — three shots you can always go to
- You arrive with an audio plan — which mic, which scenario, pre-shoot checklist
- You arrive with an anxiety protocol — specific scripts for specific triggers

## Quick Start

**Copy the template:**
```bash
cp -r shoot-template/ my-shoot-plan/
cd my-shoot-plan
```

**Fill in `BRIEF.md`**, then go through the 8 docs in order. Each one feeds into the next.

Or — if your AI assistant has this toolkit loaded — just describe what you want to shoot and let it build the plan for you.

---

## The Toolkit Philosophy

**Planning is the work. The shoot is just execution.**

Most creators think planning is optional — something you do if you have time. The toolkit treats planning as the actual work. The shooting is the easy part once the plan exists.

**Concrete beats conceptual every time.**

"Get a wide shot" is not a plan. "Stand 8 feet from the mural, put subject 3 feet in front of it, shoot at 35mm f/4, tell them to look at the camera for 5 seconds then look away" is a plan.

**Anxiety is solved with protocols, not mindset.**

Telling yourself "don't be nervous" doesn't work. Having a script for when you freeze ("go to fallback angle 2") does.
