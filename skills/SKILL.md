# Shoot Planning Skill

> Loaded when the user asks to plan any photography or videography shoot.

## Trigger Conditions

Load this skill when the user mentions any of:
- "plan a shoot", "shoot plan", "shot list", "what to shoot"
- "location scout", "scout a location", "where should I shoot"
- "shooting in [city]" or "shoot in [city]"
- "golden hour", "best time to shoot", "lighting for a shoot"
- "camera setup", "camera settings for", "what settings should I use"
- "anxiety about a shoot", "worried about a shoot"
- "shoot brief", "pre-shoot checklist"
- Any creative content creation planning request involving photography or videography

## Workflow

### Step 1 — Identify home base vs. shoot city

Ask: *"Where are you based, and where is the shoot?"*

If the **shoot city differs from the home base**, activate the travel planning section of `docs/02-research.md` first. Plan the trip (flights, accommodation, transport) before spending time on location research — no point scouting somewhere you can't afford to reach.

### Step 2 — Fill the Brief

Copy `shoot-template/BRIEF.md` to the project. Fill it in using `docs/01-brief.md` as the guide. The brief drives everything downstream.

### Step 3 — Research the Location

Use `docs/02-research.md`. If away from home base, run the travel planning steps (Step A–C) before the location research.

### Step 4–8 — Style, Shot List, Angles, Audio, Anxiety

Follow `docs/03-style.md` through `docs/07-anxiety.md` in order.

### Execute Card

Keep this on phone for the shoot day:

**Day Before**
- [ ] All batteries charged (camera + mic)
- [ ] SD card formatted
- [ ] Kit packed
- [ ] Location saved to phone maps
- [ ] Mic test confirmed
- [ ] Shot list reviewed
- [ ] Weather checked

**Pre-Shoot (First 15 Minutes)**
1. Walk location without camera
2. Set audio first (mic → receiver LED → test clip → play back)
3. Lock camera settings (WB, ISO, frame rate, focus)
4. Wide shot first — establish location

**Shooting Order**
1. Wide establishing
2. Medium shots
3. Tight detail shots
4. Talking-to-camera / walk-and-talk (main content — before fatigue)
5. Remaining angles

**Rules**
- Review every 5th setup (10-second playback)
- Keep moving — freeze = walk to new light/wall
- No clip over 2 minutes

**Post-Shoot**
- [ ] Footage offloaded
- [ ] 60-second selects reviewed in camera
- [ ] Raw footage backed up (2 copies minimum)
- [ ] Selects pulled within 24 hours

**Debrief**
1. What worked:
2. What to change:
3. Shot I didn't get:
4. Angle that surprised me:
5. Anxiety protocol update:
6. Gear I need:
